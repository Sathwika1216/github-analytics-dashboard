
GitHub Analytics Dashboard

A GitHub user analytics application built with Next.js (App Router).

🚀 Features

- Search GitHub users by username
- Dynamic routing ("/user/[username]")
- Server-side data fetching
- Displays user profile information

🛠 Tech Stack

- Next.js
- JavaScript
- GitHub REST API

📸 How it works

1. Enter a GitHub username
2. App navigates to "/user/[username]"
3. Fetches and displays user data

▶️ Run Locally

npm install
npm run dev
